# 业务机面单采集工具 V2

这个工具用于业务机连接订单整理系统 Web 服务。

## 使用方式

1. 先启动订单整理系统的 Web 服务，默认端口为 `8000`。
2. 在业务机上运行 `business_waybill_collector_v2.exe`。
3. 在窗口里填写服务端 URL，例如 `http://192.168.1.10:8000`。
4. 点击“保存配置”，再点击“启用后台心跳”。
5. 回到 8000 前端页面，业务机会按计算机名称自动显示在线。
6. 前端页面点击“开始监听”后，工具记录本机打印组件数据库基准。
7. 前端页面点击“结束监听”后，工具只上传监听期间新增的打印记录。

## 采集模式

- 原始全量采集：上传监听期间组件数据库里的原始 `task.msg`，不解析、不提取。
- 规则过滤采集：先提取可用打印文字，再按系统 App 里维护的业务机采集规则过滤后上传。

前端页面开始监听时下发的模式优先。本地默认模式只用于显示和兜底。

## 注意

不要和旧业务机采集工具同时采同一批面单，避免服务端收到重复数据。
