# V7.8.8 任务ID保留版

发布日期：2026-06-02

## 使用目录

- `订单整理系统`：放在运行订单整理/Web服务的主机上使用。
- `业务机面单监控服务`：放在业务机上使用，负责采集打印数据库并上传面单原文。

## 本版重点

- 业务机采集服务上传时保留 `task_id`、`document_id`、`task_time`、`source_record_index`。
- 订单系统监控汇总不再把同鞋款、同规格、同尺码的面单折叠成一条。
- 监控原始表和识别表保留任务追踪字段，便于回查每条面单来源。
- 标题模式补充识别 `5.0`、`ACG`、`阿尔法` 等规则场景。
- 保留 SKU 图片批量绑定入口和导入模板能力。

## 部署提醒

- 主机替换 `订单整理系统` 目录内的三个 EXE 和随包 `data` 配置。
- 业务机替换 `业务机面单监控服务` 目录，并重新运行启动或安装脚本。
- 历史已经缺失的监控原始 Excel 不能从旧文件内恢复，需要从业务机打印数据库重新采集。
